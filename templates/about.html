I Developed this Result-Server project using Python and the FastAPI framework. The database used is SQL and the frontend was developed using HTML and CSS. The design of the frontend was inspired by Google's webpage.

Project Description :

All student results are stored in the Students database. When a student requests their results, they must first Authenticate using a One-Time-Password sent to their Email. Once authenticated, a summary of the student's results will be sent to their email in the form of a PDF from the Result-Server. Admin have authority to Access and Manage the student data in the database, but Admin must also Authenticate using a One-Time-Password sent to their email before making any Updates, Creations or Deletions.
